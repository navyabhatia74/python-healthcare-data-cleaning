# Healthcare Data Cleaning & Exploratory Data Analysis using Python

## Project Overview

This project focuses on cleaning, preprocessing, validating, and analyzing healthcare appointment data using Python. The primary objective was to simulate a real-world analytics workflow where raw datasets are transformed into structured and analysis-ready data for reporting and insights generation.

The project was built using:

* Python
* pandas
* matplotlib
* seaborn
* VS Code

---

## Project Workflow

The workflow followed in this project:

Raw Dataset → Data Inspection → Data Cleaning → Data Validation → Feature Engineering → Exploratory Data Analysis → Visualization → Clean Dataset Export

---

## Key Tasks Performed

### Data Cleaning & Preprocessing

* Missing value analysis
* Duplicate detection and removal
* Column name standardization
* Datetime conversion
* Data validation
* Structured preprocessing workflow

### Feature Engineering

Created additional analytical features such as:

* Age groups
* Appointment weekdays
* Appointment months
* Waiting days between scheduling and appointment
* Attendance categories

### Exploratory Data Analysis

Performed analysis on:

* Appointment attendance trends
* Gender-based attendance patterns
* Age-group behavior
* Weekday appointment distribution
* Waiting-time impact on attendance

### Visualization

Generated automated visualizations using matplotlib and seaborn to better understand trends and patterns within the dataset.
